How It Works

Resort Finder

The Resort Finder uses an interactive map to help you find ski resorts in New York. You can enter your location in the search bar, and it will find the closest resort to you. The map also has markers for each resort, which you can click on to see more information about the resort, such as its name, location, and a link to its website.

Gear Finder

The Gear Finder helps you find the best ski gear based on your skill level and style of skiing. You can select your skill level and type of skiing from a dropdown menu, and it will display a list of recommended skis, boots, bindings, and poles for that combination. Each piece of gear has a link to its website, where you can learn more about it and make a purchase if you wish.


Behind the Scenes

This website was built using HTML, CSS, and JavaScript. The Resort Finder uses the Leaflet.js library to create the interactive map, and PapaParse to parse the CSV data of ski resorts. The Gear Finder uses JavaScript to filter the gear data based on user input and display the results.
